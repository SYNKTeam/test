SYNK Chat Widget

Embed this chat widget on your website to provide instant support to your customers.

Installation

Add this code snippet to your website, just before the closing </body> tag:

<!-- SYNK Chat Widget -->
<script>
  window.SYNKChatConfig = {
    apiUrl: 'https://your-domain.com/api',
    position: 'bottom-right', // or 'bottom-left'
    autoOpen: false
  };
</script>
<script src="https://your-domain.com/widget.js"></script>

Configuration Options

Customize the widget by setting these options in SYNKChatConfig:

  • apiUrl - The URL of your chat server API (required)
  • position - Widget position: 'bottom-right' or 'bottom-left' (default: 'bottom-right')
  • autoOpen - Automatically open the chat on page load (default: false)

JavaScript API

Control the widget programmatically using these methods:

// Open the chat widget
window.SYNKChatWidget.open();

// Close the chat widget
window.SYNKChatWidget.close();

// Toggle the chat widget
window.SYNKChatWidget.toggle();
